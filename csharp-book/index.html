Rust for C# Programmers

面向 C# 程序员的 Rust：完整培训指南

一本面向具有 C# 经验的开发者学习 Rust 的综合指南。本指南涵盖从基础语法到高级模式的所有内容，重点关注两种语言之间的概念转变和实际差异。

课程概述

  • 选择 Rust 的理由 — 为什么 Rust 对 C# 开发者很重要：性能、安全性和正确性
  • 入门 — 安装、工具链和你的第一个程序
  • 基础构建块 — 类型、变量、控制流
  • 数据结构 — 数组、元组、struct、集合
  • 模式匹配和 enum — 代数数据类型和穷尽匹配
  • 所有权和借用 — Rust 的内存管理模型
  • 模块和 crate — 代码组织和依赖
  • 错误处理 — 基于 Result 的错误传播
  • Trait 和泛型 — Rust 的类型系统
  • 闭包和迭代器 — 函数式编程模式
  • 并发 — 具有类型系统保证的无畏并发，async/await 深入
  • Unsafe Rust 和 FFI — 何时以及如何超越安全 Rust
  • 迁移模式 — 现实世界的 C# 到 Rust 模式和渐进式采用
  • 最佳实践 — 面向 C# 开发者的惯用 Rust

自学指南

本材料既适用于讲师指导的课程，也适用于自学。如果你是独自学习，以下是如何充分利用它的方法。

进度建议：

章节主题建议时间检查点
1–4设置、类型、控制流1 天你可以用 Rust 编写 CLI 温度转换器
5–6数据结构、enum、模式匹配1–2 天你可以定义带数据的 enum 并对其穷尽 match
7所有权和借用1–2 天你可以解释为什么 let s2 = s1 会使 s1 失效
8–9模块、错误处理1 天你可以创建一个多文件项目，用 ? 传播错误
10–12Trait、泛型、闭包、迭代器1–2 天你可以将 LINQ 链转换为 Rust 迭代器
13并发和 async1 天你可以用 Arc<Mutex<T>> 编写线程安全的计数器
14Unsafe Rust、FFI、测试1 天你可以通过 P/Invoke 从 C# 调用 Rust 函数
15–16迁移、最佳实践、工具链根据你的节奏参考资料 — 在编写实际代码时查阅
17综合项目1–2 天你有一个可以获取天气数据的可用 CLI 工具

如何使用练习：

  • 章节包含可折叠 <details> 块中的动手练习及解答
  • 在查看答案之前，务必先尝试练习。 与 borrow checker 的斗争是学习的一部分 — 编译器的错误信息是你的老师
  • 如果你卡住超过 15 分钟，展开解答，学习它，然后关闭它并从头再试一次
  • Rust Playground 让你无需本地安装即可运行代码

难度指示器：

  • 🟢 初学者 — 直接从 C# 概念转换
  • 🟡 中级 — 需要理解所有权或 trait
  • 🔴 高级 — 生命周期、async 内部或 unsafe 代码

当你遇到困难时：

  • 仔细阅读编译器错误信息 — Rust 的错误非常有帮助
  • 重读相关章节；像所有权（第 7 章）这样的概念通常在第二遍时会豁然开朗
  • Rust 标准库文档 非常出色 — 搜索任何类型或方法
  • 对于更深入的 async 模式，请参阅配套的 Async Rust 培训
