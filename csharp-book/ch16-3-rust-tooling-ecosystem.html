Rust for C# Programmers

C# 开发者的基本 Rust 工具

你将学到： Rust 的开发工具映射到它们的 C# 等价物——Clippy（Roslyn 分析器）、 rustfmt（dotnet format）、cargo doc（XML 文档）、cargo watch（dotnet watch）和 VS Code 扩展。

难度： 🟢 初学者

Clippy：你的自动化代码审查员

# 在项目中运行 Clippy
cargo clippy

# 将警告视为错误（CI/CD）
cargo clippy -- -D warnings

# 自动修复建议
cargo clippy --fix

#![allow(unused)]
fn main() {
// Clippy 捕获数百种反模式：

// Clippy 之前：
if x == true { }           // 警告：与 bool 的相等性检查
let _ = vec.len() == 0;    // 警告：改用 .is_empty()
for i in 0..vec.len() { }  // 警告：改用 .iter().enumerate()

// Clippy 建议之后：
if x { }
let _ = vec.is_empty();
for (i, item) in vec.iter().enumerate() { }
}

rustfmt：一致格式化

# 格式化所有文件
cargo fmt

# 检查格式化而不改变（CI/CD）
cargo fmt -- --check

# rustfmt.toml —— 自定义格式化（类似 .editorconfig）
max_width = 100
tab_spaces = 4
use_field_init_shorthand = true

cargo doc：文档生成

# 生成并打开文档（包括依赖）
cargo doc --open

# 运行文档测试
cargo test --doc

#![allow(unused)]
fn main() {
/// 计算圆的面积。
///
/// # 参数
/// * `radius` - 圆的半径（必须非负）
///
/// # 示例
/// ```
/// let area = my_crate::circle_area(5.0);
/// assert!((area - 78.54).abs() < 0.01);
/// ```
///
/// # Panics
/// 如果 `radius` 为负则 panic。
pub fn circle_area(radius: f64) -> f64 {
    assert!(radius >= 0.0, "radius must be non-negative");
    std::f64::consts::PI * radius * radius
}
// `/// ``` 块中的代码在 `cargo test` 期间编译和运行！
}

cargo watch：自动重建

# 文件更改时重建（类似 dotnet watch）
cargo watch -x check          // 仅类型检查（最快）
cargo watch -x test           // 保存时运行测试
cargo watch -x 'run -- args'  // 保存时运行程序
cargo watch -x clippy         // 保存时 lint

cargo expand：查看宏生成什么

# 查看 derive 宏的展开输出
cargo expand --lib            // 展开 lib.rs
cargo expand module_name      // 展开特定模块
