Type-Driven Correctness in Rust

Rust 中的类型驱动正确性

演讲者简介

  • 微软 SCHIE（硅与云硬件基础设施工程）团队首席固件架构师
  • 行业资深专家，专长于安全、系统编程（固件、操作系统、虚拟机监控程序）、CPU 和平台架构以及 C++ 系统
  • 2017 年开始使用 Rust 编程（@AWS EC2），从此爱上了这门语言

一本实用指南，介绍如何使用 Rust 的类型系统让整类 bug 无法编译。虽然配套的 Rust 模式 书籍涵盖了机制（trait、关联类型、type-state），但本指南展示了如何将这些机制应用到现实世界的领域 — 硬件诊断、加密、协议验证和嵌入式系统。

这里的每个模式都遵循一个原则：将不变量从运行时检查推入类型系统，让编译器强制执行。
