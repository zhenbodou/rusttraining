Rust Engineering Practices — Beyond cargo build

Rust 工程实践 — 超越 cargo build

演讲者介绍

  • 微软 SCHIE（硅与云硬件基础设施工程）团队首席固件架构师
  • 行业资深专家，专长于安全、系统编程（固件、操作系统、虚拟机监控程序）、CPU 和平台架构以及 C++ 系统
  • 2017 年开始使用 Rust 编程（@AWS EC2），从此爱上了这门语言

一份实用指南，涵盖大多数团队发现太晚的 Rust 工具链特性： 构建脚本、交叉编译、基准测试、代码覆盖率，以及使用 Miri 和 Valgrind 进行安全验证。每一章都使用来自 真实硬件诊断代码库的具体示例 — 一个大型多 crate 工作空间 — 因此每项技术都能直接映射到生产代码。

如何使用本书

本书专为自学或团队研讨会而设计。每章基本独立 — 按顺序阅读或跳转到你需要的主题。
