Rust for Python Programmers

变量和可变性

你将学到： 默认不可变变量、显式 mut、原始数值类型 vs Python 的任意精度 int、 String vs &str（早期最难的概念）、字符串格式化，以及 Rust 要求的类型注解。

难度： 🟢 初学者

Python 变量声明

# Python —— 一切可变，动态类型
count = 0          # 可变，类型推断为 int
count = 5          # ✅ 可以
count = "hello"    # ✅ 可以 —— 类型可以改变！（动态类型）

# "常量" 只是约定：
MAX_SIZE = 1024    # 没有什么能阻止 MAX_SIZE = 999 以后

Rust 变量声明

#![allow(unused)]
fn main() {
// Rust —— 默认不可变，静态类型
let count = 0;           # 不可变，类型推断为 i32
count = 5;            # ❌ 编译错误：不能两次赋值给不可变变量
count = "hello";      # ❌ 编译错误：期望整数，找到 &str

let mut count = 0;       # 显式可变
count = 5;               # ✅ 可以
count = "hello";      # ❌ 仍然不能改变类型

const MAX_SIZE: usize = 1024; # 真常量 —— 由编译器强制执行
}

Python 开发者的心智转变

#![allow(unused)]
fn main() {
// Python：变量是指向对象的标签
// Rust：变量是命名存储位置，OWN 它们的值

// 变量遮蔽 —— Rust 特有，非常有用
let input = "42";              # &str
let input = input.parse::<i32>().unwrap();  # 现在是 i32 —— 新变量，同名
let input = input * 2;         # 现在是 84 —— 另一个新变量

// 在 Python 中，你只需重新赋值并丢失旧类型：
input = "42"
input = int(input)   # 同名，不同类型 —— Python 也允许这个
但在 Rust 中，每个 `let` 创建一个真正的新绑定。旧的消失了。
}

实用示例：计数器

# Python 版本
class Counter:
    def __init__(self):
        self.value = 0

    def increment(self):
        self.value += 1

    def get_value(self):
        return self.value

c = Counter()
c.increment()
print(c.get_value())  # 1

// Rust 版本
struct Counter {
    value: i64,
}

impl Counter {
    fn new() -> Self {
        Counter { value: 0 }
    }

    fn increment(&mut self) {     # &mut self = 我将修改这个
        self.value += 1;
    }

    fn get_value(&self) -> i64 {  # &self = 我只读取这个
        self.value
    }
}

fn main() {
    let mut c = Counter::new();   # 必须是 `mut` 才能调用 increment()
    c.increment();
    println!("{}", c.get_value()); # 1
}

关键差异：在 Rust 中，方法签名中的 &mut self 告诉你（和编译器） increment 修改计数器。在 Python 中，任何方法都可以修改 任何东西 —— 你必须阅读代码才知道。

# Python —— 一种整数类型，任意精度
x = 42                     # int —— 可以增长到任意大小
big = 2 ** 1000            # 仍然可以 —— 数千位
y = 3.14                   # float —— 始终 64 位

#![allow(unused)]
fn main() {
// Rust —— 显式大小，溢出是编译时/运行时错误
let x: i32 = 42;           # 32 位有符号整数
let y: f64 = 3.14;         # 64 位 float（Python 的 float 等价物）
let big: i128 = 2_i128.pow(100); # 128 位最大 —— 无任意精度
对于任意精度：使用 `num-bigint` crate

下划线提高可读性（如 Python 的 1_000_000）：
let million = 1_000_000;   # 与 Python 相同的语法！

类型后缀语法：
let a = 42u8;              # u8
let b = 3.14f32;           # f32
}

大小类型（重要！）

#![allow(unused)]
fn main() {
// usize 和 isize —— 指针大小整数，用于索引
let length: usize = vec![1, 2, 3].len();  # .len() 返回 usize
let index: usize = 0;                     # 数组索引始终是 usize

在 Python 中，len() 返回 int，索引是 int —— 无区别。
在 Rust 中，混合 i32 和 usize 需要显式转换：
let i: i32 = 5;
let item = vec[i];    # ❌ 错误：期望 usize，找到 i32
let item = vec[i as usize]; # ✅ 显式转换
}

类型推断

#![allow(unused)]
fn main() {
// Rust 推断类型但它们是固定的 —— 不是动态的
let x = 42;          # 编译器推断为 i32（默认整数类型）
let y = 3.14;        # 编译器推断为 f64（默认 float 类型）
let s = "hello";     # 编译器推断为 &str（字符串切片）
let v = vec![1, 2];  # 编译器推断为 Vec<i32>

你始终可以显式：
let x: i64 = 42;
let y: f32 = 3.14;

与 Python 不同，类型在推断后永远不能改变：
let x = 42;
x = "hello";      # ❌ 错误：期望整数，找到 &str
}

String 类型：String vs &str

这是 Python 开发者最大的惊喜之一。Rust 有两种主要 字符串类型，而 Python 只有一种。

Python 字符串处理

# Python —— 一种字符串类型，不可变，引用计数
name = "Alice"          # str —— 不可变，堆分配
greeting = f"Hello, {name}!"  # f-string 格式化
chars = list(name)      # 转换为字符列表
upper = name.upper()    # 返回新字符串（不可变）

Rust 字符串类型

#![allow(unused)]
fn main() {
// Rust 有两种字符串类型：

// 1. &str（字符串切片） —— 借用，不可变，如字符串数据的"视图"
let name: &str = "Alice";           # 指向二进制文件中的字符串数据
                                     最接近 Python 的 str，但它是引用

// 2. String（owned string） —— 堆分配，可增长，拥有
let mut greeting = String::from("Hello, ");  # 拥有，可以修改
greeting.push_str(name);
greeting.push('!');
greeting 现在是 "Hello, Alice!"
}

何时使用哪个？

#![allow(unused)]
fn main() {
// 这样想：
&str  = "我在看别人拥有的字符串"  （只读视图）
String = "我拥有这个字符串并可以修改它"        （拥有数据）

函数参数：优先使用 &str（接受两种类型）
fn greet(name: &str) -> String {          # 接受 &str 和 &String
    format!("Hello, {}!", name)           # format! 创建新 String
}

let s1 = "world";                         # &str 字面量
let s2 = String::from("Rust");            # String

greet(s1);      # ✅ &str 直接工作
greet(&s2);     # ✅ &String 自动转换为 &str（Deref 强制）
}

实用示例

# Python 字符串操作
name = "alice"
upper = name.upper()               # "ALICE"
contains = "lic" in name           # True
parts = "a,b,c".split(",")         # ["a", "b", "c"]
joined = "-".join(["a", "b", "c"]) # "a-b-c"
stripped = "  hello  ".strip()     # "hello"
replaced = name.replace("a", "A") # "Alice"

#![allow(unused)]
fn main() {
// Rust 等价物
let name = "alice";
let upper = name.to_uppercase();           # String —— 新分配
let contains = name.contains("lic");       # bool
let parts: Vec<&str> = "a,b,c".split(',').collect();  # Vec<&str>
let joined = ["a", "b", "c"].join("-");    # String
let stripped = "  hello  ".trim();         # &str —— 无分配！
let replaced = name.replace("a", "A");     # String

关键见解：一些操作返回 &str（无分配），其他返回 String。
.trim() 返回原始数据的切片 —— 高效！
.to_uppercase() 必须创建新 String —— 需要分配。
}

Python 开发者：这样想

Python str     ≈ Rust &str     （你通常读取字符串）
Python str     ≈ Rust String   （当你需要拥有/修改时）

经验法则：
- 函数参数 → 使用 &str（最灵活）
- struct 字段       → 使用 String（struct 拥有其数据）
- 返回值       → 使用 String（调用者需要拥有它）
- 字符串字面量     → 自动 &str

打印和字符串格式化

基本输出

# Python
print("Hello, World!")
print("Name:", name, "Age:", age)    # 空格分隔
print(f"Name: {name}, Age: {age}")   # f-string

#![allow(unused)]
fn main() {
// Rust
println!("Hello, World!");
println!("Name: {} Age: {}", name, age);    # 位置 {}
println!("Name: {name}, Age: {age}");       # 内联变量（Rust 1.58+，如 f-string！）
}

格式化说明符

# Python 格式化
print(f"{3.14159:.2f}")          # "3.14" —— 2 位小数
print(f"{42:05d}")               # "00042" —— 零填充
print(f"{255:#x}")               # "0xff" —— 十六进制
print(f"{42:>10}")               # "        42" —— 右对齐
print(f"{'left':<10}|")          # "left      |" —— 左对齐

#![allow(unused)]
fn main() {
// Rust 格式化（与 Python 非常相似！）
println!("{:.2}", 3.14159);         # "3.14" —— 2 位小数
println!("{:05}", 42);              # "00042" —— 零填充
println!("{:#x}", 255);             # "0xff" —— 十六进制
println!("{:>10}", 42);             # "        42" —— 右对齐
println!("{:<10}|", "left");        # "left      |" —— 左对齐
}

调试打印

# Python —— repr() 和 pprint
print(repr([1, 2, 3]))             # "[1, 2, 3]"
from pprint import pprint
pprint({"key": [1, 2, 3]})         # 漂亮打印

#![allow(unused)]
fn main() {
// Rust —— {:?} 和 {:#?}
println!("{:?}", vec![1, 2, 3]);       # "[1, 2, 3]" —— Debug 格式
println!("{:#?}", vec![1, 2, 3]);      # 漂亮打印 Debug 格式

要使你的类型可打印，派生 Debug：
#[derive(Debug)]
struct Point { x: f64, y: f64 }

let p = Point { x: 1.0, y: 2.0 };
println!("{:?}", p);                   # "Point { x: 1.0, y: 2.0 }"
println!("{p:?}");                     # 相同，内联语法
}

类型注解：可选 vs 必需

Python 类型提示（可选，不强制执行）

# Python —— 类型提示是文档，不强制执行
def add(a: int, b: int) -> int:
    return a + b

add(1, 2)         # ✅
add("a", "b")     # ✅ Python 不在乎 —— 返回 "ab"
add(1, "2")       # ✅ 直到运行时崩溃：TypeError

# Union 类型，Optional
def find(key: str) -> int | None:
    ...

# 泛型类型
def first(items: list[int]) -> int | None:
    return items[0] if items else None

# 类型别名
UserId = int
Mapping = dict[str, list[int]]

Rust 类型声明（必需，编译器强制执行）

#![allow(unused)]
fn main() {
// Rust —— 类型强制执行。始终。无例外。
fn add(a: i32, b: i32) -> i32 {
    a + b
}

add(1, 2);         # ✅
add("a", "b");  # ❌ 编译错误：期望 i32，找到 &str

可选值使用 Option<T>
fn find(key: &str) -> Option<i32> {
    返回 Some(value) 或 None
    Some(42)
}

泛型类型
fn first(items: &[i32]) -> Option<i32> {
    items.first().copied()
}

类型别名
type UserId = i64;
type Mapping = HashMap<String, Vec<i32>>;
}

关键见解：在 Python 中，类型提示帮助你的 IDE 和 mypy 但不影响运行时。 在 Rust 中，类型就是程序 —— 编译器使用它们保证内存安全， 防止数据竞争，并消除空指针错误。

练习

🏋️ 练习：温度转换器（点击展开）

挑战：编写一个函数 celsius_to_fahrenheit(c: f64) -> f64 和一个函数 classify(temp_f: f64) -> &'static str，根据阈值返回 "cold"、"mild" 或 "hot"。打印 0、20 和 35 摄氏度的结果。使用字符串格式化。

🔑 解答 
fn celsius_to_fahrenheit(c: f64) -> f64 {
    c * 9.0 / 5.0 + 32.0
}

fn classify(temp_f: f64) -> &'static str {
    if temp_f < 50.0 { "cold" }
    else if temp_f < 77.0 { "mild" }
    else { "hot" }
}

fn main() {
    for c in [0.0, 20.0, 35.0] {
        let f = celsius_to_fahrenheit(c);
        println!("{c:.1}°C = {f:.1}°F — {}", classify(f));
    }
}

关键要点：Rust 需要显式 f64（无隐式 int→float），for 直接迭代数组（无 range()），if/else 块是表达式。
