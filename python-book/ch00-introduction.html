Rust for Python Programmers

面向 Python 程序员的 Rust：完整培训指南

一本面向具有 Python 经验的开发者的 Rust 学习综合指南。本指南涵盖从基础语法到高级模式的所有内容，重点关注从动态类型、垃圾回收语言转向具有编译时内存安全性的静态类型系统语言时所需的概念转变。

如何使用本书

自学格式：首先完成第一部分（第 1-6 章）—— 这些章节与你已经熟悉的 Python 概念紧密对应。第二部分（第 7-12 章）介绍 Rust 特有的概念，如所有权和 trait。第三部分（第 13-16 章）涵盖高级主题和迁移。

进度建议：

章节主题建议时间检查点
1–4设置、类型、控制流1 天你可以用 Rust 编写一个 CLI 温度转换器
5–6数据结构、enum、模式匹配1–2 天你可以定义一个带有数据的 enum 并对它进行穷尽的 match
7所有权和借用1–2 天你可以解释为什么 let s2 = s1 会使 s1 失效
8–9模块、错误处理1 天你可以创建一个多文件项目，使用 ? 传播错误
10–12trait、泛型、闭包、迭代器1–2 天你可以将列表推导式转换为迭代器链
13并发1 天你可以用 Arc<Mutex<T>> 编写一个线程安全的计数器
14Unsafe、PyO3、测试1 天你可以通过 PyO3 从 Python 调用 Rust 函数
15–16迁移、最佳实践自行安排参考资料 —— 在编写实际代码时查阅
17综合项目2–3 天构建一个完整的 CLI 应用程序，将所有内容串联起来

如何使用练习：

  • 章节包含可折叠 <details> 块中的动手练习及其解答
  • 务必在展开解答之前先尝试练习。与 borrow checker 的斗争是学习的一部分 —— 编译器的错误信息就是你的老师
  • 如果卡住超过 15 分钟，展开解答，学习它，然后关闭它并从头再试一次
  • Rust Playground 让你无需本地安装即可运行代码

难度指示器：

  • 🟢 初学者 — 直接从 Python 概念转换
  • 🟡 中级 — 需要理解所有权或 trait
  • 🔴 高级 — 生命周期、async 内部机制或 unsafe 代码

当你遇到困难时：

  • 仔细阅读编译器错误信息 —— Rust 的错误提示异常有帮助
  • 重读相关章节；像所有权（第 7 章）这样的概念通常在第二遍时才会豁然开朗
  • Rust 标准库文档 非常出色 —— 搜索任何类型或方法
  • 对于更深入的 async 模式，请参阅配套的 Async Rust 培训
