Rust Patterns & Engineering How-Tos

Rust 模式与工程实践指南

演讲者介绍

  • 微软 SCHIE（硅与云硬件基础设施工程）团队首席固件架构师
  • 行业资深专家，专长于安全、系统编程（固件、操作系统、虚拟机管理程序）、CPU 和平台架构以及 C++ 系统
  • 2017 年开始使用 Rust 编程（@AWS EC2），从此爱上了这门语言

这是一本面向中高级 Rust 开发者的实用指南，涵盖真实代码库中常见的模式。这不是一本语言教程——它假设你已经能够编写基本的 Rust 代码并希望更上一层楼。每一章都聚焦于一个概念，解释何时以及为何使用它，并提供可编译的示例和 inline 练习。

目标读者

  • 已完成《Rust 程序设计语言》但在"我该如何设计这个？"问题上挣扎的开发者
  • 将生产系统从 C++/C# 移植到 Rust 的工程师
  • 在泛型、trait 约束或生命周期错误上遇到瓶颈，希望获得系统性工具包的任何人

前置要求

开始之前，你应该熟悉以下内容：

  • 所有权、借用和生命周期（基础水平）
  • enum、模式匹配和 Option/Result
  • 结构体、方法和基础 trait（Display、Debug、Clone）
  • Cargo 基础：cargo build、cargo test、cargo run

完成练习

每章都以动手练习结束。为了最大化学习效果：

  1. 先自己尝试——在查看答案前至少花 15 分钟
  2. 手写代码——不要复制粘贴；手写能建立肌肉记忆
  3. 修改解决方案——添加功能、改变约束、故意破坏某些东西
  4. 检查交叉引用——大多数练习结合了多章的模式

附录中的综合项目将全书的模式整合到一个生产质量的系统中。

目录

第一部分：类型级模式

1. 泛型全貌 🟢 单态化、代码膨胀权衡、泛型 vs enum vs trait 对象、const 泛型、const fn。

2. trait 深入讲解 🟡 关联类型、GAT、blanket impl、marker trait、vtable、HRTB、扩展 trait、enum 分发。

3. Newtype 和 Type-State 模式 🟡 零成本类型安全、编译时状态机、构建器模式、config trait。

4. PhantomData —— 不携带数据的类型 🔴 生命周期标记、单位度量模式、drop 检查、变型。

第二部分：并发与运行时

5. 通道和消息传递 🟢 std::sync::mpsc、crossbeam、select!、背压、actor 模式。

6. 并发 vs 并行 vs 线程 🟡 OS 线程、作用域线程、rayon、Mutex/RwLock/原子操作、Condvar、OnceLock、无锁模式。

7. 闭包和高阶函数 🟢 Fn/FnMut/FnOnce、闭包作为参数/返回值、组合子、高阶 API。

8. 智能指针和内部可变性 🟡 Box、Rc、Arc、Weak、Cell/RefCell、Cow、Pin、ManuallyDrop。

第三部分：系统与生产

9. 错误处理模式 🟢 thiserror vs anyhow、#[from]、.context()、? 操作符、panic。

10. 序列化、zero-copy 和二进制数据 🟡 serde 基础、enum 表示、zero-copy 反序列化、repr(C)、bytes::Bytes。

11. Unsafe Rust —— 受控的危险 🔴 五种超能力、可靠的抽象、FFI、UB 陷阱、arena/slab 分配器。

12. 宏 —— 编写代码的代码 🟡 macro_rules!、何时（不）使用宏、过程宏、derive 宏、syn/quote。

13. 测试和基准测试模式 🟢 单元/集成/文档测试、proptest、criterion、mock 策略。

14. Crate 架构和 API 设计 🟡 模块布局、API 设计清单、人性化参数、feature 标志、workspace。

15. Async/Await 要点 🔴 Future、Tokio 快速入门、常见陷阱。（深入 async 覆盖，请参阅我们的 Async Rust 培训。）
